About the project

Auzugswächter (roughly translated as "the elevator watcher" from German) is a service which monitors the status of elevators and escalators on the stations on the stations managed by the DB Station&Service AG.

You can check the status of facilities on the map, check them on Twitter from @aufzugswaechter, or subscribe to certain facility via e-mail or as push messages via Android app.

Aufzugswächter monitors status of facilities by regularly querying the FaSta API provided by the DB Station&Service AG.
If a facility changes the status, this triggers a tweet as well as e-mail and push message to users which subsribed to this facility.

For more information, please see the following presentation.
